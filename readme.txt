h1. Настройки

1. Поставить вызов cron.py по расписанию каждые 60(?) минут.
2. В методе AddDomainToControlPanel находится URL скрипта добавления домена в 
панель управления. В скрипте находятся логин и пароль для доступа к API панели 
(предполагается, что для всех панелей одного типа они одинаковые). 
3. В конструкторе агентов находится URL центра администрирования
4. Требуемые настройки на хостах: chmod через ftp должен быть разрешен, права 
на корни сайтов должны быть 042775.

h2. По расписанию

При каждом вызове cron.py происходит:
1. Генерация заданий для генерации дорвеев (Doorway, DoorwaySchedule).
2. Генерация заданий для спама по имеющимся дорвеям (SpamTask).
3. Генерация заданий для сбора сниппетов (SnippetsSet).
4. Проверка активности агентов.
